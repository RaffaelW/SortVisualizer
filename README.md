# SortVisualizer
## Description
SortVisualizer is an Android application that visualizes how sorting algorithms work.

## Implemented algorithms:
+ InsertionSort
+ ShellSort
+ SelectionSort
+ BubbleSort
+ ShakerSort
+ CombSort
+ GnomeSort
+ CycleSort
+ HeapSort
+ MergeSort
+ LSD RadixSort
+ MSD RadixSort
+ QuickSort
